PROJEK INFRASTRUKTUR & KOMPUTASI AWAN

MEMBUAT & MENGKONFIGURASI UBUNTU

Proyek instalasi dan konfigurasi sistem operasi open-source berbasis Linux (Ubuntu). Praktik ini mencakup manajemen sistem dasar, pengenalan command line interface (CLI), dan eksplorasi environment Linux.

INSTALASI DEBIAN SERVER

Melakukan instalasi distribusi Linux Debian. Fokus utama proyek ini adalah memahami proses partisi disk, pengaturan jaringan dasar, dan manajemen package melalui repositori resmi Debian.

DEPLOYMENT PRIVATE CLOUD DENGAN OWNCLOUD

Merupakan implementasi praktis dari mata kuliah Komputasi Awan (Cloud Computing). Saya membangun solusi private cloud storage menggunakan OwnCloud yang di-deploy di atas sistem operasi Ubuntu Server. Proyek ini mendemonstrasikan bagaimana data dapat dikelola, disinkronisasi, dan diakses secara aman layaknya layanan cloud komersial.
